# Project Title

Domain Manager

## Description

Simple CRUD to domains.

## Getting Started

### Dependencies

Back
* php 8.1.9
* laravel 9.43

Front
* Vuejs 3.2.45

### Installing

* Clone the repository
* go to the `back-laravel` folder
* run `composer install`
* configure your database
* run `php artisan serve`
* go to the `front-vuejs`
* run `npm install`
* run `npm run dev`
